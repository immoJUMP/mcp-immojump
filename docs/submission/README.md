# Submission package — Anthropic Software Directory

Every file in this folder is a **template**. Fields marked `«TODO»` or
check-boxes marked `[?]` must be verified by the vendor. Do not submit
the connector to Anthropic until those placeholders are resolved.

| File | Purpose |
|------|---------|
| `SUBMISSION_CHECKLIST.md` | Form-field mapping + review-criteria sign-off |
| `PRIVACY_POLICY.md` | Template for the public Privacy Policy URL |
| `DEMO_ACCOUNT.md` | Test-account setup, seed-data recipe, reviewer prompts |
| `LANDING_PAGE.md` | Copy for the public landing page |
| `branding/` | Logo + screenshot assets — create and populate before submitting |

## What Claude Code cannot do

The directory is self-serve, but the submission form at
<https://clau.de/mcp-directory-submission> is a manual web flow. It
requires human input for test-account passwords, uploaded branding, and
policy-acceptance checkboxes. Use this folder as the launchpad, then open
the form in a browser to submit.

Queue time varies. Escalate to `mcp-review@anthropic.com` only after 10
business days.
